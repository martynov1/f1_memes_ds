# f1_memes_ds

Бот для автоматического кросспоста мемов с [r/formuladank](https://reddit.com/r/formuladank) в канал на Discord-сервере.

## Что делает

- Забирает топовые мемы с r/formuladank через RSS (без API-ключей Reddit)
- Сначала берёт мемы за день, если новых нет — за неделю
- Постит от старых к новым, чтобы лента шла хронологически
- Отправляет в Discord через вебхук — с картинкой, заголовком и ссылкой на оригинал
- Не дублирует — помнит уже опубликованные посты
- Оптимизирует ссылки на изображения для полного разрешения

## Как запустить

1. Создайте виртуальное окружение и установите зависимости:

   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install requests python-dotenv
   ```

2. Создайте `.env` файл с вашим Discord-вебхуком:

   ```bash
   DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/...
   ```

3. Запустите:

   ```bash
   venv/bin/python rdt_to_ds.py
   ```

## Деплой

Проект настроен на автоматический деплой через GitHub Actions — при пуше в `main` изменения подтягиваются на сервер по SSH. На сервере скрипт запускается ежедневно через cron.
