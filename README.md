# f1_memes_ds

Бот для автоматического кросспоста мемов с [r/formuladank](https://reddit.com/r/formuladank) в канал на Discord-сервере.

## Что делает

- Забирает топовые мемы за день с r/formuladank через RSS (без API-ключей Reddit)
- Постит их в Discord-канал через вебхук — с картинкой, заголовком и ссылкой на оригинал
- Не дублирует — помнит уже опубликованные посты
- Оптимизирует ссылки на изображения для полного разрешения

## Как запустить

1. Установите зависимости:

   ```bash
   pip install requests
   ```

2. Укажите URL вашего Discord-вебхука в `rdt_to_ds.py`
3. Запустите:

   ```bash
   python rdt_to_ds.py
   ```

4. Для ежедневного запуска добавьте в cron или launchd

## Приходите на наш Discord-сервер [Просмотр F1](https://discord.gg/fGeu3jd7)

Если вы фанат Формулы 1 и любите качественные мемы — заходите к нам. У нас свежие мемы с r/formuladank появляются автоматически, а ещё живое комьюнити для обсуждения гонок, драм и всего, что происходит в паддоке. Box, box, box!
